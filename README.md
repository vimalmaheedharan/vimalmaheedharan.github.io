# Engineering at Scale - Portfolio Platform

A professional, high-performance, responsive portfolio platform tailored for showcasing software architecture, cloud-native engineering solutions, and observability systems.

## Tech Stack
* **HTML5** (Semantic Architecture)
* **Tailwind CSS** (Utility-First Design)
* **Custom Vanilla CSS** (Micro-interactions, glassmorphism, background particle fields)

## Project Structure
Mapped exactly to match the deployment asset structure required for clean cross-linking between top-level landing pages and profile subdirectories.
